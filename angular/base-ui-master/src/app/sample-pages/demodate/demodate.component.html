Custom Datepicker

Single Datepicker


Definition & Usage

Urp datepicker function base on the configuration

    Definition of every attribute
  1. type = type of daterange for example single datepicker or daterangepicker
  2. formControlName = for variable name
  3. lable = Datepicker lable name
  4. submitted = submitted the form
  5. required = It’s a mandatory field, for example required="true" mandatory if required="false" is not a mandatory
  6. minDate = enabling the minimum select date
  7. maxDate = enabling the maximum select date
  8. mode = display the months or years only
  9. format = formatting the date value

HTML tag

            <ui-date  
                formControlName="variable_name" label='Datepicker' [submitted]="submitted"
                required="true" [maxDate]="maxDate" [minDate]="minDate" format='DD/MM/YYYY'">
            </ui-date>

select year

            <ui-date  
                type="datepicker"
                formControlName="variable_name" label='Datepicker' [submitted]="submitted"
                required="true" [maxDate]="maxDate" [minDate]="minDate" format='YYYY' mode = 'year'>
            </ui-date>

select month

            <ui-date  
                formControlName="variable_name" label='Datepicker' [submitted]="submitted"
                required="true" [maxDate]="maxDate" [minDate]="minDate" format='MM/YYYY' mode = 'month'>
            </ui-date>

typescript

            submitted: boolean = false;
            minDate: Date = new Date('1921-01-01');
            maxDate: Date = new Date();

Time Picker


Definition & Usage

    Definition of Timepicker attributes
  1. showMeridian = for formatting the time, if showMeridian is true timepicker follow the 12 hours format and if showMeridian is false timepicker follow the 24 hours format
  2. showSeconds = enabling the second input field

HTML tag

                    <ui-time  
                    formControlName="variable_name" [showMeridian]= "true" [showSeconds]= "true">
                    </ui-time>
