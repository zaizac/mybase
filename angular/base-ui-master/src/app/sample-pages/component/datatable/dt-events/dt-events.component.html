Row clicking event

When clicking at the row (other than checkbox), it will trigger datatable row selecting event

Datatable will emit this event through editRecord. To listen to this event, simply write a function to handle this event

editRecord will return row data. e.g: In example below, we use the row data to alert message

In template:

            <ui-datatable [dataValues]='dtData4' [tableInit]='headerConfig10' (editRecord)='handleRowClick($event)'>
            </ui-datatable>

In controller:

            handleRowClick(row) {{ '{' }}
              alert(`${{ '{' }}row.name{{ '}' }} is working in ${{ '{' }}row.company{{ '}' }}`);
            {{ '}' }}
