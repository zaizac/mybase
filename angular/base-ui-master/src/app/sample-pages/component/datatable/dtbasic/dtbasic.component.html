Urp Datatable has two basic input properties.

  1. tableInit: Datatable initialiasation instance. Datatable can be configured through this property. This is required for server side pagination. <ui-datatable [tableInit]="dataConfig"></ui-datatable>
  2. dataValues: The data source for the table. This property only required for Frontend pagination. For server side pagination, this is not required. <ui-datatable [dataValues]="data"></ui-datatable>

No Data:

If there is no data inside the table, it will display as below

Frontend Pagination:

The example below is the most basic configuration for Urp Datatable. we pass a list of data to dataValues property. This is a frontend pagination

In example below, we has a list of data assigned to variable dtData. Then we pass dtData to datatable dataValues property 

      In TS file, set the dataValues, e.g: 
      dtData = [
        {{
          '{'
        }} name: 'Ethel Price', gender: 'female', company: 'Johnson, Johnson and Partners, LLC CMP DDC', age: 22m {{
          '}'
        }},
        {{ '{' }} name: 'Claudine Neal', gender: 'female', company: 'Sealoud', age: 55{{ '}' }}
      ]
<ui-datatable [dataValues]="dtData" > </ui-datatable >
    If no configuration supply, Urp Datatable will have following properties
  • It will take object key as header
  • Column will be based on object key and value
  • 15 rows will be displayed per page
  • All columns are sortable

Urp Datatable can also be used multiple times in single component (this is the 2nd table)

Serverside Pagination:

For Serverside pagination, datatable configuration need to be initialiased and assigned to Urp Datatable 'tableInit' property

For basic configuration, service api call need to be done in tableInit data method. This method will return an observable to datatable and contains a callback function with datatable request payload as parameter which can be used for service payload e.g: 

          dtConfig = {{ '{' }}
            data: request => {{ '{' }}
              return mockServerSidePaginationResponse({{ '{' }}
                start: request.start,
                length: request.length,
                draw: request.draw
              {{ '}' }});
          {{ '}' }}
<ui-datatable [tableInit]="dtConfig"></ui-datatable>
