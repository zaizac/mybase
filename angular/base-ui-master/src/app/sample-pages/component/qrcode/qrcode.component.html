QR Code Generate

 
          <ui-qrcode [elementType]="img" [value] = "value" errorCorrectionLevel = "M"
            version = "5"></ui-qrcode>

Definition properties:

[elementType]

ElementType for generating QRcode base on the which elements. for example img or URL or canvas

For example:
[elementType]="img" OR
[elementType]="url" OR
[elementType]="canvas"

[value]

passing the value for generate QRCode

For example:
[value]="value"

errorCorrectionLevel

Error correction capability allows to successfully scan a QR Code even if the symbol is dirty or damaged. Four levels are available to choose according to the operating environment.
Level L (Low) 7% of data bytes can be restored.
Level M (Medium) 15% of data bytes can be restored.
Level Q (Quartile) 25% of data bytes can be restored.
Level H (High) 30% of data bytes can be restored.

For example:
errorCorrectionLevel = "L" OR
errorCorrectionLevel = "M" OR
errorCorrectionLevel = "Q" OR
errorCorrectionLevel = "H"

version

indicating the overall dimensions of the QRCode. total version 1 to 40.
For more information and QRCode design refer here https://en.wikipedia.org/wiki/QR_code#Storage

For example:
version = "1" OR
version = "5" OR
version = "25" OR
version = "40"
