Definition and Usage

The tag defines a horizontal tab in an HTML page.

To define a clickable tab header button, and tags are required.

To define the tab content, and tags are required.

Attributes

Attribute Value Description th:classappend customtab Specifies the tab as custom tab
Attribute Value Description href URL / Tab Content ID Specifies the URL of the tab content to be open th:classappend active Specifies the tab as active
Attribute Value Description id Specifies the unique id of the tab content th:classappend active Specifies the tab as active
