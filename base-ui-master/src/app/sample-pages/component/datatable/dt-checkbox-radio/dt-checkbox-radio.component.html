Checkbox

A column can be defined as checkbox by setting the type value as checkbox when initialise tableInit columns

To select a row, we need to click on the checkbox itself. The selected data can be retrieved by calling table init selectedData attribute. The attribute will return an array. If no checkbox checked, it will return null value. The checkbox can be placed in any column.

            this.headerConfig = {{ '{' }}
              columns: [
                {{ '{' }} field: 'SlNo' {{ '}' }}, 
                {{ '{' }}
                  type: 'checkbox'
                {{ '}' }}, 
                {{ '{' }} field: 'name' {{ '}' }}, 
                {{ '{' }} field: 'gender' {{ '}' }}, 
                {{ '{' }} field: 'company' {{ '}' }} 
              ],
            {{ '}' }}

To retrieve the data, just call tableInit selected data. e,g: this.headerConfig.selectedData

selected - {{ this.headerConfig.selectedData | json }}

Radiobutton

A column can be defined as radio button by setting the type value as radio when initialise tableInit columns

To select a row, we need to click on the radio button itself. The selected data can be retrieved by calling table init selectedRowData attribute. The attribute will return the selected row data object. If no radio button checked, it will return null value. The radio button can be placed in any column

              this.headerConfig = {{ '{' }}
                columns: [
                  {{ '{' }}
                    type: 'radio'
                  {{ '}' }}, 
                  {{ '{' }} field: 'SlNo' {{ '}' }}, 
                  {{ '{' }} field: 'name' {{ '}' }}, 
                  {{ '{' }} field: 'gender' {{ '}' }}, 
                  {{ '{' }} field: 'company' {{ '}' }} 
                ],
              {{ '}' }}

Selected row data: {{ headerConfig2.selectedRowData | json }}

Important

When using both checkbox and radio on same table, the radio button will not be rendered. Because only 1 selection type can be used per table. If both type defined in the column, only checkbox column will render because it take precedent over radio button

              this.headerConfig = {{ '{' }}
                columns: [
                  {{ '{' }} field: 'SlNo' {{ '}' }}, 
                  {{ '{' }}
                    type: 'checkbox'
                  {{ '}' }}, 
                  {{ '{' }} field: 'name' {{ '}' }}, 
                  {{ '{' }} field: 'gender' {{ '}' }}, 
                  {{ '{' }} field: 'company' {{ '}' }} 
                ],
              {{ '}' }}
