Attribute Type Value
docMgtId string document Id
*Mandatory if init viewOnly = true 
[docMgtId]="docMgtId"
formControl FormControl Transaction document 
[formControl] = "file"
refDoc FileConfig RefDocuments
*Mandatory if init viewOnly = false 
[refDoc]="refDoc"
init FileUploadConfig
  • type : string ‣ default value == 'file'
  • viewOnly : boolean ‣ default value == false
  • textAlign : string ‣ default value == 'center'
  • width : string ‣ default value == '100%'
  • height : string ‣ default value == '100%'
  • radius : string ‣ default value == '4px'
  • autoUpload : boolean ‣ default value == false
  • cropper : boolean ‣ default value == false
  • noImgSrc : string ‣ default value == 'assets/default/no_image.jpg'
